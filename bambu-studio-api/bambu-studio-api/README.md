# Bambu Studio API

Headless Bambu Studio CLI exposed as a REST API. Designed as the
`bambu-studio-api` sidecar described in the
[Bambuddy slicer-api docs](https://wiki.bambuddy.cool/features/slicer-api/),
packaged as a Home Assistant App.

See the [repository README](https://github.com/griffinmartin/ha-app-bambu-studio-api)
for install, configuration, and disclaimer details.

See [DOCS.md](./DOCS.md) (also rendered in the add-on's **Documentation** tab
inside HA) for runtime configuration and troubleshooting.
